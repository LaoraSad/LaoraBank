# Student Average Calculator

## Description

This project is a simple Python program that calculates the average score of a student.  
The program is an educational example used to demonstrate basic programming concepts such as variables, input, conditions, and calculations.

## How it works

1. The program asks the user to enter the student's name.
2. It requests three scores: speaking, listening, and reading.
3. The program calculates the average score using a simple formula.
4. It checks the value of the average.
5. The program prints the level of the student: High, Medium, or Low.

## Status

> The project is currently running as a basic learning example for programming and English explanation practice.
